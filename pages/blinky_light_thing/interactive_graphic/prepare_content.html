A session starts by making an attestation from an account that is provably yours - this is some form of digital identity that uses a blockchain. The attestation must include the blink rate you are using and just a few other elements that make it easier to decode. We've predefined a schema on Ethereum Attestation Services to make this easier for you - it looks like this:


fullName : "James Canterbury"; optional
messageColor: "BLUE"; recommended
terminationColor: "RED"; recommended
blinkRate: "200"; in milliseconds required
description: "some extra text here" optional

The attestation returns a 64 character hexidecimal string which both serves as a pointer to the attestation and a unique ID that would be impossible to determine ahead of time.

Tip

Creating the attestation using your wallet proves that it is the authentic you (assuming you control your keys and have used some form of proof of personhood). The UID of the attestation uses the time stamp of the Ethereum block in which the attestation is posted - proving that your session is contemporaneous.
